---
title: "Get Multiple Products"
openapi: "GET /products/{ids}"
---

Retrieve detailed information about multiple products by specifying a comma-separated list of product IDs. This endpoint returns an array of product objects with comprehensive data about each product.

## Path Parameters

<ParamField path="ids" type="string" required>
  Comma-separated list of product IDs (e.g., "123,456,789")
</ParamField>

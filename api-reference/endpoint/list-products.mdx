---
title: "List Products"
openapi: "GET /products"
---

Retrieve a paginated list of products. This endpoint returns an array of product objects with comprehensive data about each product.

## Query Parameters

<ParamField query="offset" type="integer" default="0">
  Number of items to skip for pagination
</ParamField>

<ParamField query="limit" type="integer" default="10" min="1" max="100">
  Maximum number of items to return per page
</ParamField>

## Welcome to Pure's API

The Pure Public API provides access to marketplace data including products, spot prices, and more. This API allows you to integrate Pure's precious metals marketplace data into your applications.

## Getting Started

To start using the Pure API, you'll need to:

<Steps>
  <Step title="Request an API Key">
    Contact us at [collectpure.com](https://www.collectpure.com) to request an
    API key for your application.
  </Step>
  <Step title="Include Your API Key">
    Add your API key to the `x-api-key` header in all your requests.
  </Step>
  <Step title="Make Your First Request">
    Try the [Authentication Test](/api-reference/endpoint/protected) endpoint to
    verify your API key is working.
  </Step>
</Steps>
